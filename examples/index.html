Fit

Fit will try to fit $image inside $thumb. If $thumb is larger than $image it will center it.
If $thumb is thinner than $image then it will be centered vertically, otherwise it will be centered horizontaly. 

$image = new File('my-picture.jpeg');
$thumb = new TrueColor(200, 300);
$thumb->fit($image);
